# Honeypot Website 2.0

## Setup

### Install nvm

https://github.com/creationix/nvm#install--update-script


### Install nvm at the specific version of node

```bash
nvm install 10.10.0
```

### Install dependencies

```bash
npm install
```

### Run app

```bash
npm run develop
```
